  • TX_BUSY [ro]: Indicates whether the transmitter is busy (1) or ready to send data (0)
  • RX_NOT_EMPTY [ro]: Indicates whether the receive FIFO (currently single-element) contains data received

  • DATA [rw]: Read to pop byte from the receive FIFO (requiring RX_NOT_EMPTY == 1). Write to send a character (requiring TX_BUSY == 0). The transmission is always 8-bit.
